## 定位（HBuilder的h5+有自动定位到当前的功能  不过需要连接手机）

```
 html,body, #container{width: 100%;height: 100%;}//记得要给高度百分百
<div id="container"></div>

<script src="https://webapi.amap.com/maps?v=1.4.10&key=您申请的key值"></script>
这个key值要自己申请

<script>
	var map = new AMap.Map('container', {//这个container就是容器
		resizeEnable: true, //是否监控地图容器尺寸变化
		zoom:11, //初始化地图层级
		center: [116.397428, 39.90923] //初始化地图中心点
	});
</script>
```
