## 服务器代理

```
在根目录新建一个 vue.config.js   记住  改完一定要重启服务器  npm run  serve

例如要请求的地址  "https://www.duitang.com/napi/index/hot/?start=0&include_fields=sende
module.exports = {
	devServer: {
		proxy: {
			'/api': {
				target: 'https://www.duitang.com',
				changeOrigin: true,
				// secure: false,
				pathRewrite: {
					'^/api': ''
				}
			}
		} 
	}
}

/api就想当于是https://www.duitang.com  所以拼接起来就是了
 $.ajax({
        url: "/api/napi/index/hot/?start=0&include_fields=sende",
        type: "get",
        success(a) {
          console.log(a)
        }
});
```

## 在后端解决跨域问题

```
在前后端通过express框架进行路由交互的时候   把这段复制到后端js  就不用考虑跨域了
app.all('*', function (req, res, next) {
    res.header("Access-Control-Allow-Origin", "*");
    res.header("Access-Control-Allow-Headers", "Content-Type,Content-Length, Authorization, Accept,X-Requested-With");
    res.header("Access-Control-Allow-Methods", "PUT,POST,GET,DELETE,OPTIONS");
    res.header("X-Powered-By", ' 3.2.1')
    if (req.method == "OPTIONS") {
        res.send(200);/*让options请求快速返回*/
    } else {
        next();
    }
});
```
